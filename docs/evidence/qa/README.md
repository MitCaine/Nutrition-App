# Manual QA evidence

`nutrition-app-qa-test-specification.docx` is a point-in-time pre-RC manual QA worksheet. Its
partial results and informal observations preserve the state of that test session; they are not a
current defect list or release claim.

Use the later [Release Candidate 1 QA report](../../rc1-release-qa.md) and the current
[Testing Guide](../../testing.md) for the repository's release evidence and automated validation
boundaries. The DOCX is retained unchanged as historical evidence.
